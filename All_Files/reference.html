Element: <!--...-->

Definition:

Used to construct comments not displayed by the browser.

Syntax:

<!-- comments here -->

Example:

<!--
The comment tag is usually used by website developers to write notes, documentation, and to-dos within their code.
-->
<!--
The comment tag can be used for creating if statements in some browsers. E.g
-->
<!--[if gte Chrome 42]>
<p>Awesome you are using the Chrome version greater than 41.</p>
<![endif]-->
<!--
That's it for the comment tag
!-->
